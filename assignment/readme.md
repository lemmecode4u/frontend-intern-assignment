# Mini Product Dashboard

## Overview

This project is a small frontend web application that displays product data fetched from an API. The app allows users to search products by name, filter by category, and sort by price. Additionally, a product detail modal can be opened by clicking on any product.

## Features

- **Search** by product name.
- **Filter** products by category.
- **Sort** products by price (Low to High / High to Low).
- **Responsive Design** for mobile, tablet, and desktop.
- **Product Detail Modal**: View detailed information about a product when clicked.
  
## Technologies Used

- HTML
- CSS (Flexbox/Grid for layout)
- JavaScript (Vanilla)
- Fetch API

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/frontend-intern-assignment.git
    ```

2. Navigate to the project directory:
    ```bash
    cd frontend-intern-assignment
    ```

3. Open `index.html` in your browser to view the app.
